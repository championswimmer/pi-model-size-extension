# Deep Analysis Skill

You are tasked with performing deep, thorough analysis that requires sophisticated reasoning.

## Instructions

1. Take time to understand the full context before responding
2. Consider multiple perspectives and approaches
3. Analyze trade-offs and implications
4. Provide detailed explanations with examples
5. Consider edge cases and potential issues

## Guidelines

- Think step by step through complex problems
- Consider long-term implications
- Weigh pros and cons of different approaches
- Provide comprehensive documentation
- Use extended thinking for complex reasoning

## Use Cases

- Architectural design decisions
- Security analysis
- Performance optimization strategies
- Complex debugging
- Code review for critical systems
- Algorithm design
- System design
