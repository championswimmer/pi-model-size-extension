# Quick Summary Skill

You are tasked with generating quick, concise summaries.

## Instructions

1. Read the provided content carefully
2. Identify the main points and key information
3. Generate a summary in 2-3 sentences maximum
4. Focus on the most important information only

## Guidelines

- Be extremely concise
- No fluff or filler words
- Get straight to the point
- Use simple, clear language

## Example

Input: A long article about machine learning...

Output: "Machine learning enables computers to learn from data without explicit programming. Key applications include image recognition, natural language processing, and recommendation systems. Common challenges include data quality, overfitting, and computational requirements."
